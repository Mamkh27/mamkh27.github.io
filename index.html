EEC Smart Toaster System

Embedded Systems

With the use of software and hardware components, we built an embedded toaster system on the TI CC3200 LaunchPad interface. On trigger, the system turns on the conveyer belt which then dispenses a slice of bread into the toaster. We thoroughly designed, developed, implemented, and tested embedded software in support of project design. This project required the connection to Wi-Fi and Amazon Web Services to publish and subscribe to IoT topics. We were able to successfuly authenticate users and attach policies over AWS to establish a connection with iOS application in XCode. The tools and languages used in this project include: Github, TI CC3200 LaunchPad, AWS, CCS, Swift, C, DTMF-tone decoding, GPIO-pin handling.
